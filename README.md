# 📺 YouTube Clone (Frontend Practice)

Hi there! My name is Andranik, I am 15 years old, and this is my custom frontend clone of the YouTube homepage interface. This project was built completely from scratch to level up my web development and design skills.

## 🚀 About The Project
This is a pixel-perfect layout of the YouTube main page. The entire core structure, video grid, and navigation layout were coded **100% manually**.

### 🛠 Built With
* **HTML5** — Semantic page structure.
* **CSS3** — Custom styling, dark mode theme, and layout design.
* **CSS Flexbox** — Alignment for the top header navigation and sidebar menus.
